Chapter 2. Starting eXeLearning

Windows

To start the eXe, double-click the eXe icon that should have appeared on your desktop after installation. If you don't have an eXeLearning icon on your desktop you will need to find the application's icon in your Windows start menu under Programs. You should probably maximise the eXeLearning window to utilise all the space you have available in which to work. eXeLearning portable is a new version without installation

Linux

If you installed one of the standard packages, you should find eXeLearning in your system menu (education).Â  Alternatively you can start a shell in a terminal window and type exe (all lower case) from the command line. This will launch eXe. eXeLearning portable is a new version without installation

Mac OS X

Double click the eXeLearning application icon.
